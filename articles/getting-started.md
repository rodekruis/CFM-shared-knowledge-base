# Regional Feedback Management Platform User Guide

### What is this document about?

Standard Operating Procedure (SOP) on how to navigate the platform. 

### Why is this important?

The Regional Feedback Management Platform is accessed by different roles. It is important that a National Society onboarded into the platform is able to access the platform correctly and that its users (depending on their roles) are able to fulfil activities.

### 1. Accessing the Platform

- Logging into the Shared Feedback Management Platform
- You can log in using the email and password that you have set up.

### 2. Logging into the platform for the first time

When a National Society is onboarded into the platform, the Feedback Manager user will be created by the Administrator.

- Administrator creates a user for the Feedback Manager
- Feedback Manager successfully logs in the Platform
- Feedback Manager can then create new 'User Access Requests' for the rest of the National Society members that require access to the Platform.

<ins>*FOR FEEDBACK MANAGER ROLE*</ins>

The Feedback Manager Role in the National Society will ONLY be able to create new user access requests. This is done by:

1. Clicking on the 3 dots on the top left of the page
2. Selecting 'User Access Request'
3. Clicking on the '+Create User Access Request' button that is located on the top right of the page
